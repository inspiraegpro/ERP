# Inspera ERP (Wrapstyle ERP) 🚀

نظام تخطيط موارد المؤسسات (ERP) متكامل ومبني خصيصاً لإدارة الأعمال التجارية والخدمية (مثل خدمات السيارات)، يوفر إدارة شاملة للمخازن، المبيعات، المشتريات، الحسابات المالية، والموارد البشرية في مكان واحد.

## ✨ المميزات الرئيسية (Key Features)

- **📦 إدارة المخازن (Inventory Management):** 
  - تتبع دقيق للرولات والقطع والبواقي.
  - جرد وإدارة الأرصدة (رصيد أول المدة، إذن صرف، إذن استلام).
  - دعم المقاسات (الطول، العرض، المساحة) وربط الرولات المفتوحة.
- **💰 المبيعات والمشتريات (Sales & Purchases):**
  - إصدار فواتير المبيعات والمشتريات وإدارتها.
  - ربط الفواتير بأوامر التشغيل (Service Jobs).
- **📊 الحسابات المالية (Accounting & Treasury):**
  - شجرة حسابات متكاملة، قيود يومية تلقائية.
  - إدارة الخزينة والبنوك ومتابعة المصروفات والإيرادات.
- **👥 الموارد البشرية (HR):**
  - إدارة بيانات الموظفين وتتبع الرواتب.
- **🚗 خدمات السيارات (Car Services):**
  - ربط الخدمات وأوامر الشغل بنوع السيارة والجزء المراد العمل عليه (مثل مقاسات النوافذ والأبواب).

## 🛠️ التقنيات المستخدمة (Tech Stack)

- **الواجهة الخلفية (Backend):** Node.js, Express.js
- **قاعدة البيانات (Database):** Local File Database (JSON-based / FileDatabaseManager)
- **الواجهة الأمامية (Frontend):** HTML5, CSS3, Vanilla JavaScript
- **أخرى:** Multer (لرفع الملفات)، JsonWebToken (للمصادقة)، Cors

## 🚀 طريقة التشغيل (Installation & Setup)

تأكد من تثبيت [Node.js](https://nodejs.org/) على جهازك، ثم اتبع الخطوات التالية:

1. **استنساخ المشروع (Clone the repository):**
   ```bash
   git clone https://github.com/YourUsername/inspera_erp.git
   cd inspera_erp
   ```

2. **تثبيت الحزم المعتمدة (Install dependencies):**
   ```bash
   npm install
   ```

3. **تشغيل السيرفر (Start the server):**
   ```bash
   node server.js
   ```

4. **الوصول إلى النظام (Access the App):**
   - افتح المتصفح على الرابط: `http://localhost:13620`
   - بيانات الدخول الافتراضية (إذا تم إعدادها):
     - **اسم المستخدم:** admin | **كلمة المرور:** admin123
